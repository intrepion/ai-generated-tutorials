# Getting Started: Building Your First "Hello World" with Astro 🚀

If you are looking for a way to build a fast, modern website, you’ve likely heard of **Astro**.

Astro is a web framework designed for speed. It does this by delivering "Zero JavaScript" by default, meaning it only ships the code your users actually need to see. But before we dive into the complex stuff like Content Collections or Image Optimization, let’s start at the very beginning.

In this guide, we are going to build the most minimal "Hello World" project possible using Astro.

## Prerequisites

Before we start, make sure you have **Node.js** installed on your machine (version 18.14.1 or higher is recommended).

Even though the code looks like plain HTML, Astro is doing some heavy lifting behind the scenes:
