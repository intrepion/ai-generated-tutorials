**Minimal Astro Hello World Tutorial**

**1. Create Project**

Bash
```
npm create astro@latest
```

Choose "minimal" (empty) template. Name it e.g. hello-astro.⁠Docs.astro

**2. Navigate & Run**

Bash
```
cd hello-astro
npm run dev
```

Open http://localhost:4321.

**3. Edit src/pages/index.astro**

astro
```
---
---
<h1>Hello World from Astro!</h1>
```

That's the most minimal setup. Static HTML by default.⁠Docs.astro
